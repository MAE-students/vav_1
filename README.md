# [Vibration Analysis] – Open Knowledge Base

This repository is part of the community-driven knowledge base for the **MSc in Music and Acoustic Engineering – Politecnico di Milano**.
Its goal is to **collect, preserve, and continuously improve** notes, summaries, and code for this specific course.

## 🏗️ Organization Structure
To keep materials organized and easy to manage, we use **one repository per subject**.

- Browse the repository list to find the specific exam or course you are looking for.
- All repositories are public and accessible to everyone.

## 🤝 How to Contribute
This project follows an **Open Source / Open Knowledge** model.
Since base permissions are set to **Read**, contributions happen through **Issues**.

If you:

- find an error
- want to suggest an improvement
- want to add new content
- want to request clarification

➡️ **Open an Issue in the repository of that course**.

In the Issue you can:

- clearly describe the problem or suggestion
- indicate the file and section involved
- attach files or screenshots
- propose new material

The maintainers will take care of integrating the changes.

📄 A detailed guide on how to write a proper Issue is available in the [dedicated markdown](CONTRIBUTING.md).

## 📝 Editing with Overleaf
Each course repository can be imported into **Overleaf** and edited in LaTeX with full GitHub synchronization.

This works:

- whether you fork the repository or not
- even if you only have read permissions (local edits)

Typical workflow:

1. Import the GitHub repository into Overleaf
2. Edit the notes in LaTeX from Overleaf
3. Changes are synced back to GitHub automatically (if you have write access, such as from a fork)
